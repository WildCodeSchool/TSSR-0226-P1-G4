# Installation de KeePass sur WIN01 




&nbsp;

 **Étape 1 — Lancement de l'installateur KeePass**
 Télécharger KeePass depuis le site officiel keepass.info. Double-cliquer sur le fichier `.exe` pour lancer l'assistant d'installation.

&nbsp;
 

<img width="1597" height="372" alt="Screenshot 2026-03-18 152920" src="https://github.com/user-attachments/assets/2b7ff90b-c009-4ee2-aaa9-d1190d0e0473" />




&nbsp;

 **Étape 2 — Installation terminée**
 Valider les étapes de l'assistant et cliquer sur "Terminer". KeePass est maintenant installé sur WIN01 et prêt à être configuré.

&nbsp;

# Installation du logiciel WinSCP

&nbsp;

 **Étape 1 — Téléchargement de WinSCP**
 Se rendre sur winscp.net et télécharger la dernière version stable. WinSCP permet le transfert de fichiers sécurisé entre Windows et Linux via le protocole SFTP/SSH.

&nbsp; 

<img width="2535" height="1685" alt="Capture d&#39;écran 2026-03-19 102720" src="https://github.com/user-attachments/assets/6543e0ff-cef1-4b28-9fbe-df8b54639531" />

&nbsp;

 **Étape 2 — Lancement de l'assistant d'installation**
 Double-cliquer sur le fichier `.exe` téléchargé pour démarrer l'assistant d'installation WinSCP.

&nbsp;

<img width="703" height="590" alt="Screenshot 2026-03-19 101819" src="https://github.com/user-attachments/assets/98032d98-15b8-48a2-8308-a513da3a1483" />

&nbsp;

 **Étape 3 — Acceptation de la licence**
 Lire et accepter le contrat de licence utilisateur final (EULA) pour continuer l'installation.

&nbsp; 

<img width="780" height="528" alt="Screenshot 2026-03-19 101946" src="https://github.com/user-attachments/assets/54414327-8038-4761-a3c9-b54f88f948f4" />

&nbsp;

 **Étape 4 — Choix du type d'installation**
 Sélectionner "Installation typique" pour une configuration standard incluant tous les composants nécessaires.

&nbsp;

<img width="783" height="527" alt="Screenshot 2026-03-19 102459" src="https://github.com/user-attachments/assets/cecdf9e7-1f6c-46da-9c2e-81401b4e542f" />

&nbsp;

 **Étape 5 — Choix du style d'interface**
 Choisir le style "Commander" pour afficher deux panneaux côte à côte (local et distant), ce qui facilite le transfert de fichiers.

&nbsp;
<img width="634" height="459" alt="Screenshot 2026-03-19 102513" src="https://github.com/user-attachments/assets/2f207fcb-904e-4bb7-8d19-8709b98d3327" />
&nbsp;

 **Étape 6 — Installation en cours**
 L'assistant copie les fichiers nécessaires sur le système. Patienter jusqu'à la fin du processus.

&nbsp;
<img width="457" height="366" alt="Screenshot 2026-03-19 102533" src="https://github.com/user-attachments/assets/8b7dbcbc-545e-4016-b907-b33139824a15" />
&nbsp;

 **Étape 7 — Ouverture de WinSCP**
 WinSCP est installé. Au premier lancement, la fenêtre de connexion s'affiche automatiquement.

&nbsp;
<img width="1342" height="753" alt="Screenshot 2026-03-19 102757" src="https://github.com/user-attachments/assets/c542ba88-9789-4087-899f-e11d8ff3d841" />

&nbsp;

 **Étape 8 — Configuration de la connexion vers SRVLX01**
 Renseigner les informations de connexion :
 - Protocole : SFTP
 - Hôte : `172.16.10.6`
 - Port : `22`
 - Utilisateur : `wilder`

&nbsp;
<img width="1346" height="754" alt="Screenshot 2026-03-19 102826" src="https://github.com/user-attachments/assets/59474d67-4b54-4694-9d42-e4e56f9c587d" />
&nbsp;

 **Étape 9 — Saisie du mot de passe**
 Entrer le mot de passe du compte wilder pour établir la connexion SSH vers SRVLX01.
